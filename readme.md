# TalentFlow

TalentFlow is a web application designed to manage the workflow of professional internships between students.

The goal of the system is to centralize and simplify the management of internship students, allowing the registration of relevant information, document management, and structured tracking of each student.

The application is built using **Laravel, React, and Inertia.js**, combining the power of Laravel on the backend with a modern user experience on the frontend.

---

## Technologies Used

### Frontend
